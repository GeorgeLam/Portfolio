Covid-19 Mobility Data

Using data from Google's Coronavirus Global Mobility Report to display changes in activity across a range of location types. Project uses Node and Express. The data is stored using MongoDB and displayed with chart.js.

Github

See more...

Since the global outbreak of Covid-19, daily life has been impacted in all parts of the world. Data released by Google for over 140 countries, demonstrates the change in activity at certain location types (i.e. workplaces, recreational areas).

The data was released in csv form (obtainable here), with data divided on a national and regional level. I downloaded the csv data, uploaded it to a MongoDB, and then created an API that fetches the data for the country, region, and location type requested by the user. Chart.js is used to display the data.
